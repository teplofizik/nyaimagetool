# Инструкция NyaImageTool

## Запуск
Использование: ./NyaImageTool.exe <пусть к скрипту>
Загрузка и сохранение образов идёт в самом скрипте.

## Команды
Подключение доп. файла скрипта:
```
include <filename>
```

Загрузка из образа FIT:
```
load <filename>
```

Запись в образ FIT (если загружено из FIT, или же загружены ядро, рамфс, девтри и настроены их параметры загрузки)
```
store <filename>
```

Добавление файла:
```
file <path> <localfilename> <mode> <user> <group>
```

Если надо только обнеовить содержимое, без смены прав:
```
file <path> <localfilename>
```

Добавление папки:
```
dir <path> <mode> <user> <group>
```

Добавление ссылки:
```
slink <path> <target> <mode> <user> <group>
```

Удаление файла или папки:
```
rm <path>
```
